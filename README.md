# 导航
[预览链接](https://miffy24.github.io/nav-demo/index.html)
## 主要功能
1. 键盘按键电脑键的a-z,可以去对应的网址。
2. hover按钮显示E，编辑功能，编辑跳转网址。
3. 头部有个搜索框，点击对应按钮可以分别去百度和Google搜索
## 使用技能
1. html5和css3
页面的布局，采用了flex布局。
2. 原生js 
    - 键盘按钮采用原生js生成，避免span标签的空隙
    - 监听用户事件
    - 监听input事件
